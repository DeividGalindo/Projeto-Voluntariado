Bem-vindo à Plataforma de Conexão Voluntária!

Esta plataforma foi criada para aproximar ONGs de voluntários. Nosso objetivo é otimizar o processo de captação de voluntários, tornando-o mais eficiente e transparente.

Navegue até a página "Ver Necessidades" para encontrar uma oportunidade ou cadastre uma nova demanda se você representa uma instituição.
